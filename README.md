## Grading
```Final grade = 0.7*HW + 0.3*Exam```

* `HW` - Average grade for the assignments 1 to 5. 
You can get extra points by solving HW 6, but no more than 10 in total. Namely, `HW = (hw1 + hw2 + hw3 + hw4 + hw5 + hw6)/5`


* `Exam` -  Grade for the exam.
